## Run Locally

Clone the project

```bash
  git clone https://github.com/devpenzil/devpenzil-website.git
```

Go to the project directory

```bash
  cd devpenzil-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
